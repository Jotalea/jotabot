Available commands

j!ban: Bans a user from the server. Syntax (numbers are placeholders for the user ID): j!ban user=123456789, reason="No spamming"

j!config: Get the link for the config site (W.I.P.).

j!credits: Shows a list with people that helped making the bot.

j!help: Shows this list of commands.

j!invite: Get the invite for Jota-Bot.

j!kick: Kicks a user from the server. Syntax (numbers are placeholders for the user ID): j!kick user=123456789

j!ping: Get the bot's latency.

j!rr: Play the russian roulette: if you lose, you get banned.

j!say: Makes the bot say anything.

j!shutdown: Turns off the bot (Only for Jotalea).

j!ssh: Open an SSH-like interface on Discord, where you can run real commands (Unstable, work in progress).

j!uptime: Displays how much time has the bot been working.

j!version: Get the current bot version.

j!web: Get a link to Jotalea's website.
